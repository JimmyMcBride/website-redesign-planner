# Impeccable Context

## Design Context

- Build a client decision app, not the final production marketing site.
- Audience: non-technical stakeholders choosing a website direction from research, audit findings, design systems, and homepage demos.
- Visual target: polished, clear, market-specific, and evidence-led. Avoid generic Tailwind defaults.
- Interaction target: design-system switching must feel immediate and consistent across examples.
- Verification target: run Svelte checks, static build, Playwright screenshots, and mobile/desktop browser review before handoff.
